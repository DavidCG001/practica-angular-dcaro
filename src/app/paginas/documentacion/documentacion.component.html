Documentación de la API

Guía para acceder y utilizar la API de Monster Hunter World.

¿Cómo acceder a la API?

Para acceder a la base de datos de Monster Hunter World, utilizamos una API REST que ofrece una gran variedad de datos sobre armas, armaduras, monstruos y más. Aquí te explicamos cómo hacer peticiones para obtener la información que necesitas.

Endpoints Disponibles

  • Armaduras: /armor
    Obtén información sobre las armaduras disponibles en Monster Hunter World. Puedes buscar armaduras por tipo, rango o nombre.
  • Armas: /weapons
    Accede a información detallada sobre las armas, incluyendo el nombre, tipo, ataques, y otros atributos.

Cómo Realizar una Petición

Para hacer una solicitud, puedes usar herramientas como Postman, o realizar peticiones HTTP directamente en tu código.

Ejemplo de Petición:

  GET https://mhw-db.com/armor

Este endpoint te devolverá una lista de todas las armaduras disponibles en Monster Hunter World.

Filtrar Resultados:

Puedes añadir parámetros de búsqueda para filtrar los resultados, por ejemplo: 

  GET https://mhw-db.com/armor/leather

Esto solo devolverá las armaduras de tipo "helmet". Puedes buscar por cualquier tipo de armadura, arma o monstruo que desees.

Errores Comunes

Si recibes un error, esto es lo que puedes esperar:

  • 400 - Bad Request: La petición no es válida. Revisa los parámetros.
  • 404 - Not Found: No se encontraron resultados para tu búsqueda.
  • 500 - Internal Server Error: Hubo un problema con el servidor. Intenta más tarde.

¿Dónde Encontrar Más Información?
