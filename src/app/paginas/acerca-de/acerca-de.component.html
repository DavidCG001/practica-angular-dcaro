Acerca de esta aplicación

Una aplicación para explorar el mundo de Monster Hunter World.

¿Qué es Monster Hunter?

Monster Hunter es una serie de videojuegos desarrollada por Capcom. En estos juegos, los jugadores asumen el rol de cazadores que deben derrotar a gigantescos monstruos en una variedad de misiones. En esta aplicación podrás explorar diferentes armas, armaduras y más, todos basados en Monster Hunter World.

¿Qué hace esta aplicación?

Esta aplicación proporciona una base de datos sobre las armas y armaduras de Monster Hunter World. Puedes buscar diferentes tipos de armaduras y armas, obtener sus detalles, estadísticas y mucho más. El objetivo es ayudarte a mejorar tu experiencia en el juego y facilitar la información que necesitas para tu progreso.

¿Cómo funciona?

La aplicación utiliza una API externa que recopila datos del juego y los presenta en un formato fácil de usar. Puedes realizar búsquedas, filtrar resultados, y explorar las distintas categorías para encontrar lo que más te interese sobre armas, armaduras y otros elementos del juego.

¿Quién lo desarrolló?

Este proyecto fue desarrollado por un fan de Monster Hunter como parte de un ejercicio para aprender Angular y cómo consumir APIs. La intención es crear una herramienta útil para los jugadores que busquen información rápida sobre sus armas y armaduras favoritas.
